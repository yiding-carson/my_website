Pearson correlation coefficent

The hypothesis test for my investigation is the pearson correlation coeficient(r),and the statistical hypotheses:

\[H_0: r = 0 \]

\[H_A: r \ne 0\]

Linear regression model

Previous Pearson correlation efficient hypotheses test has reavealed there is a statistically significant correlation between infant mortality and GDP,then a linear regression statictic hypothesis with four assumption are applied in the invistigation. As a conclusion,a linear reggression model was fitted to predict the dependent variable, infant mortality, using measures of GDP as predictor. The overall regression model was statistically significant, F(1,222)=88.04, p<.001, and explained 28.4% of the variability in infant mortality, R2=.284. The estimated regression equation was infant mortality=4966.89-0.284GDP. The negative slope for GDP was statistically significant, b=0.609, t(222)=17.9, p<.001, 95% CI [4422.76,5511.03]. Final inspection of the residuals supported normality and homoscedasticity.

The linear regression stastistic hypothesis: \[H_0: r = The data do not fit the linear regression model \]

\[H_A: r \ne The data fit the linear regression model\]

The linear regression equation: \[y=+x+\]
